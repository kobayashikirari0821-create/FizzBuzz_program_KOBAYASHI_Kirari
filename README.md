# FizzBuzz_program_KOBAYASHI_Kirari
株式会社フォーク様との最終面接に向け作成した、「FizzBuzz問題」の解答

## 使用した言語
JavaScript

## 制作および登録するまでのプロセス
1. GitHubにて新規アカウントを開設
2. 新規リポジトリを作成
3. GitHubのWebブラウザ上で"FizzBuzz.js"という新規ファイルを作成
4. JavaScriptを用いて、以下の条件を満たすように1から20までの数を表示するスクリプトを記述
  - 3の倍数であり、かつ5の倍数であるときは"FizzBuzz"と表示
  - 3の倍数である（かつ5の倍数でない）ときは"Fizz"と表示
  - 5の倍数である（かつ3の倍数でない）ときは"Buzz"と表示
  - 上記のどれにも当てはまらないときはそのままの数を表示
5. 編集内容を保存
6. Google Chromeの開発者ツールのConsoleタブから、記述したスクリプトの動作をテストし、要件を満たしていることを確認
